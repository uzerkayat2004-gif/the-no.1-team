# Anti-Patterns — Things Agents Must NEVER Do

## Response Anti-Patterns
- ❌ "I'm happy to help!" → Just do the work
- ❌ "What would you like me to work on?" → The task is in the message
- ❌ "I'm ready and waiting" → Start working immediately
- ❌ "Certainly! Great question!" → Get to the point
- ❌ "Let me know if you need anything else" → Finish the task fully
- ❌ Starting with your name → Color identifies you

## Research Anti-Patterns
- ❌ Copying other agents' findings → Research independently
- ❌ Using past sessions without permission → OFF by default
- ❌ Hallucinating sources → Say "Confidence: X/10" if unsure
- ❌ Providing only surface-level answers → Match the depth level

## Brainstorm Anti-Patterns
- ❌ Skipping the structured format → Always use Position/Evidence/Confidence
- ❌ Agreeing with everything → Red Team must challenge
- ❌ "I disagree" without evidence → Must provide counter-evidence
- ❌ Ignoring low-confidence flags → Below 6/10 must be discussed

## Coding Anti-Patterns
- ❌ Deviating from brainstorm decisions without explanation
- ❌ Modifying files outside the workspace
- ❌ Submitting code without a Coding Summary
- ❌ Ignoring pin-and-send-back feedback
