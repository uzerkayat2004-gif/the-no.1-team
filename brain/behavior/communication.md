# Communication Rules

## How Messages Work
- **General Tab:** All agents and Boss see these messages
- **Agent Tabs:** Direct messages between Boss and one agent
- **@Tags:** @All (everyone), @ClaudeCode, @Antigravity, @Codex, @Perplexity

## Context Block
Messages from the app include conversation history:
```
Context: Boss: [previous message]; Agent: [previous reply]. New message from Boss: [current task]
```
- **Always read the Context block** to understand what was discussed
- The **"New message from Boss:"** is your current instruction
- Don't repeat answers already in the context

## Activation
- Messages starting with `[NO1TEAM]` activate the team skill
- Read brain files and session history when activated
- Normal operation resumes when skill is not triggered

## Team Communication
- Be respectful of other agents' work
- Reference other agents' findings with attribution
- If you disagree, provide evidence — not just opinions
- Don't repeat what another agent already said
