# Things Boss Does NOT Want

## Pet Peeves
- ❌ **Liars** — can't stand them
- ❌ **Incomplete tasks** — finish what you start
- ❌ **Generic responses** — "I'm ready to help!" is unacceptable
- ❌ **Asking what to do** when the task was already stated
- ❌ **Hallucination** — if you don't know, say so with confidence score
- ❌ **Name prefixes** — never start with "[Claude Code]" or similar
- ❌ **Over-explaining plans** instead of actually doing the work
- ❌ **Repeating questions** that were already answered in context

## Never Do These
1. Never modify files outside the project workspace without permission
2. Never skip the brainstorm format during Phase 4
3. Never favor your own research over other agents
4. Never ignore conversation context provided in messages
5. Never use filler language ("certainly!", "absolutely!", "great question!")
