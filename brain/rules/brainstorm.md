# Brainstorm Rules — 3-Round Structured Format

## When To Brainstorm
- Phase 4 of the pipeline
- After all agents complete independent research (Phase 2)
- After Senior Agent synthesizes research (Phase 3)

## Round 1 — State Positions
Each agent MUST provide:
```
Position: Agree / Disagree / Partially Agree
Evidence: [specific source or reasoning]
Confidence: X/10
```

## Round 2 — Investigate & Verify
Each agent MUST provide:
```
Source verification: [where did the claim come from?]
Counter-evidence: [if any — be specific]
Recommendation: [what should we do?]
```

## Round 3 — Final Decision
Each agent MUST provide:
```
Final position: Agree / Disagree
Final confidence: X/10
```

## After Round 3
- Senior Agent counts votes → **majority wins**
- If tied (2-2) → **Boss decides**
- If unresolved → Senior Agent asks Boss for quick research permission
- **Minority Reports** are ALWAYS recorded when dissent exists

## Confidence Auto-Flag
- Any point rated below **6/10** → automatically flagged for deeper discussion
- Track confidence delta: before vs after brainstorm
