# The 7-Phase Research Pipeline

## Phase 0 — Task Creation
Boss posts the task on the General Tab. All agents see it.

## Phase 1 — Pre-flight
- Agents ask clarifying questions
- Agents confirm they understand the task
- Boss assigns research platforms and depth level
- Boss designates Senior Agent (or auto-assign by task type)

## Phase 2 — Independent Research
- All agents research IN PARALLEL, INDEPENDENTLY
- No agent may reference another agent's findings
- Sync barrier: ALL agents must finish before Phase 3
- Each finding rated with confidence score (1-10)
- Below 6/10 → auto-flagged for brainstorm discussion

## Phase 3 — Synthesis
- Senior Agent combines ALL research into ONE document
- This is done LIVE — Boss watches in real-time
- Senior Agent must attribute each finding to its source agent
- Conflicts between agents are highlighted, not hidden

## Phase 4 — Brainstorm
- Point-by-point discussion
- 3 structured rounds (Position → Verify → Decide)
- Red Team challenges every point with evidence
- If unresolved after 3 rounds → Boss decides

## Phase 4B — Deep Research Trigger
- If a brainstorm point needs more data, Senior Agent can request deep research
- Only the relevant agent goes deep; others continue brainstorming
- Results fed back into the brainstorm

## Phase 5 — Consensus
- Majority vote wins
- Ties → Boss breaks the tie
- Minority reports ALWAYS recorded (dissent is valued)
- All agents must state final position + confidence

## Phase 6 — Final Document
- Senior Agent produces the final polished document
- All agents + Boss rate session quality (1-10)
- Senior Agent updates brain memory
- Session saved to brain/memory/sessions/
