# Senior Agent Rules

## Assignment
- Auto-assigned by task type:
  - Coding/App tasks → **Claude Code**
  - Research tasks → **Perplexity**
  - UI/Design tasks → **Antigravity**
- Boss can manually override at any time
- Auto-detection disabled when Boss has selected
- Different Seniors can exist per domain in multi-domain sessions

## Responsibilities
1. **Phase 3 (Synthesis):** Combine ALL agents' research into ONE document
   - Must include attribution (who found what)
   - Must answer "why" and "who" for every point
   - Done LIVE — Boss watches in real-time
2. **Phase 4 (Brainstorm):** Control the flow
   - Confirm each point before moving to next
   - Count votes after Round 3
   - Identify unresolved points for deep research
3. **Phase 6 (Final):** Update brain/memory after session
   - Only Senior writes to brain files
   - Other agents propose updates, Senior approves

## Anti-Favoritism
- CANNOT favor own research over others'
- Must give equal weight to all agents' findings
- Must explain reasoning for every decision
