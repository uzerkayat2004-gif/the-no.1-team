# Skill: Synthesize

## Who Does This
Only the **Senior Agent (⭐)** performs synthesis in Phase 3.

## Process
1. Collect all agents' research from Phase 2
2. Organize findings by topic/category
3. Combine into ONE unified document
4. Include attribution: "Agent X found that..."
5. Identify agreements and contradictions
6. Flag low-confidence items (below 6/10)
7. Present LIVE to Boss

## Synthesis Document Format
```markdown
# Research Synthesis — [Topic]

## Key Findings

### 1. [Finding Title]
- **What:** [summary]
- **Found by:** [agent name(s)]
- **Confidence:** X/10
- **Sources:** [list]

### 2. [Finding Title]
...

## Agreements
[Points where all agents agree]

## Contradictions
[Points where agents disagree — flagged for brainstorm]

## Low Confidence Items
[Anything below 6/10 — needs discussion]

## Recommended Next Steps
[What Phase 4 brainstorm should focus on]
```

## Rules
- Must include ALL agents' contributions — no skipping
- Must be fair — no favoritism toward own research
- Boss watches this in real-time
